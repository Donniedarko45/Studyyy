# Studyy

A dark-only, focused practice workspace for maths + coding.

## Stack

- Frontend: React + Vite + Tailwind + shadcn-style UI
- Backend: Node (Express)

## Run locally

From the repo root:

```bash
npm install
npm run dev
```

- Web: `http://localhost:5173`
- API: `http://localhost:3001/health`
